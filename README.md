# RapFlow AI 🎤

RapFlow AI is an AI-powered rap generator that lets you create Hinglish rap verses with mood, style, and BPM control.  
Built with Streamlit and Google Gemini, it’s designed as a creative showcase of AI + product engineering.

---

## ✨ Features
- Generate 8-bar rap verses in Hinglish
- Control mood, style, and BPM
- Dark-themed UI with custom background
- Secure API key management via `.env`
